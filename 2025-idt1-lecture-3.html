Exploring RFID with Arduino

Arduino IDE

  1. Includes the libraries providing commands to communicate with the RFID reader.
  2. Defining variables
    • Defines the pin numbers used for the RFID reader.
    • Defines the pin number for an LED.

Arduino IDE

    Setup block

  1. Starts serial communication.
  2. Prints a message to the Serial Monitor.
  3. Sets the LED pin (pin 2) as an output.

Arduino IDE

    Loop block

  1. Checks if a new RFID card is within the reader's range using the NOT operator (!).
  2. Sets the pinLED (pin 2) to HIGH, turning it on.
  3. Loops through each byte of the UID, prints each byte in hexadecimal format.
  4. Pauses for 1 second, before turning off the LED.
